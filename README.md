# Personal Portfolio Website

The website contains information about me as a Data Science and  Analysis student. It also includes some of my technical skills, selected projects, and contact information.

## Website Sections

- Home
- About Me
- Skills
- selected Projects
- Contact

## Technologies Used

- HTML
- CSS


## Website Design

The website uses a pastel purple theme with:

- A custom top banner
- A background image
- Navigation links between sections

## Project Files

- `index.html` — contains the structure and content of the website(text).
- `style.css` — contains the colors, fonts, and design.
- `banner.jpg` — the top banner background image.
- `background.png` — the background image used behind the website sections.
